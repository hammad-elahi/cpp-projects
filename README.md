# C++ Projects 🖥️

A collection of personal C++ projects built during my early learning journey. Each project was created to practice a specific concept — not just understand it theoretically, but actually build something real with it.

## The Approach
No tutorials. No passive learning. Just picking up a concept, understanding it, and immediately building a project around it. Each project stacks on top of the previous one.

## Projects

| Project | Concepts Practiced |
|---|---|
| ATM Simulation | Conditionals, loops, control flow |
| Scientific Calculator | Functions, modular design |
| Class Performance Analyzer | Arrays, sorting, searching |
| Password & Text Analyzer | Strings, char arrays, ASCII logic |
| Dynamic Contact Manager | Pointers, dynamic memory, heap allocation |

## Note
These are early learning projects — built for understanding, not perfection. The progression from project 1 to 5 tells the story better than anything else.
